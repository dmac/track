# Track

A basic time tracking CLI utility.

# Usage

    $ track start myproj
    $ track note myproj starting project
    $ track note myproj reticulating splines
    $ track stop myproj
    $ track show myproj
    myproj
            2014-01-01 01:02:03 - 2014-01-01 02:03:04
                    starting project
                    reticulating splines
